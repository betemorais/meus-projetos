Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase tenho um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande (tag <big>) e um texto pequeno (tag <smal>) nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído (tag <del>) para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido (tag <ins>) para dar ênfase e indicar que ele foi adicionado depois.

Existe também existe o sublinhado com a tag <u> (não semântica)

Texto sobescrito

Para inserir coisas do tipo x20+3 (tag <sup>)

Texto subescrito

Para inserir coisas do tipo H2O (tag <sub>)
